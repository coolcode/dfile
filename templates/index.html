Sample use cases

How to upload

# Uploading is easy using curl
$ curl --upload-file ./hello.txt https://dfile.app/hello.txt
https://dfile.app/66nb8/hello.txt

$ curl -H "Max-Downloads: 1" -H "Max-Days: 5" --upload-file ./hello.txt https://dfile.app/hello.txt
https://dfile.app/66nb8/hello.txt
# Download the file
$ curl https://dfile.app/66nb8/hello.txt -o hello.txt

Add an alias to .bashrc or .zshrc [gist]

# Add this to .bashrc or its equivalent
transfer() { if [ $# -eq 0 ]; then echo -e "No arguments specified. Usage:\necho transfer /tmp/test.md\ncat /tmp/test.md | transfer test.md"; return 1; fi
tmpfile=$( mktemp -t transferXXX ); if tty -s; then basefile=$(basename "$1" | sed -e 's/[^a-zA-Z0-9._-]/-/g'); curl --progress-bar --upload-file "$1" "https://dfile.app/$basefile" >> $tmpfile; else curl --progress-bar --upload-file "-" "https://dfile.app/$1" >> $tmpfile ; fi; cat $tmpfile; rm -f $tmpfile; }

# Now you can use transfer command
$ transfer hello.txt
More examples

Upload multiple files at once


$ curl -i -F filedata=@/tmp/hello.txt -F filedata=@/tmp/hello2.txt https://dfile.app/

# Combining downloads as zip or tar archive
$ curl https://dfile.app/(15HKz/hello.txt,15HKz/hello.txt).tar.gz
$ curl https://dfile.app/(15HKz/hello.txt,15HKz/hello.txt).zip

Encrypt your files before the transfer

# Encrypt files with password using gpg
$ cat /tmp/hello.txt|gpg -ac -o-|curl -X PUT --upload-file "-" https://dfile.app/test.txt

# Download and decrypt
$ curl https://dfile.app/1lDau/test.txt|gpg -o- > /tmp/hello.txt

Scan for malware

# Scan for malware or viruses using Clamav
$ wget http://www.eicar.org/download/eicar.com
$ curl -X PUT --upload-file ./eicar.com https://dfile.app/eicar.com/scan

# Upload malware to VirusTotal, get a permalink in return
$ curl -X PUT --upload-file nhgbhhj https://dfile.app/test.txt/virustotal

Backup mysql database, encrypt and transfer

# Backup, encrypt and transfer
$ mysqldump --all-databases|gzip|gpg -ac -o-|curl -X PUT --upload-file "-" https://dfile.app/test.txt

Send email with transfer link (uses alias)

# Transfer and send email with link (uses alias)
$ transfer /tmp/hello.txt | mail -s "Hello World" user@yourmaildomain.com

Using Keybase.io

# Import keys from keybase
$ keybase track [them] # Encrypt for recipient(s)
$ cat somebackupfile.tar.gz | keybase encrypt [them] | curl --upload-file '-' https://dfile.app/test.txt # Decrypt
$ curl https://dfile.app/sqUFi/test.md |keybase decrypt

wget uploads also supported

# wget
$ wget --method PUT --body-file=/tmp/file.tar https://dfile.app/file.tar -O - -nv

Transfer pound logs

# grep syslog for pound and transfer
$ cat /var/log/syslog|grep pound|curl --upload-file - https://dfile.app/pound.log
